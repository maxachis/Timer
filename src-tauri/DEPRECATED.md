# Deprecated: Tauri implementation

This directory contains the original Tauri 2 (Rust) backend for Timer. It has been superseded by the Wails 2 (Go) implementation in `../wails-app/` (see commit `dba58cf`).

**Status:** frozen. Kept for reference only.

- Do not add new features here.
- Bug fixes should go to `wails-app/` instead.
- The root-level `src/` Svelte frontend and root `package.json` are part of this deprecated Tauri stack. The active frontend lives at `wails-app/frontend/`.

This directory may be removed in a future cleanup.
